UI

Les fondamentaux et tendances du design

0. C'est quoi l'UI Design ?

Les bases visuelles

Le designer UI (User Interface) s'occupe de tous les éléments graphiques que l'utilisateur voit.

  • Les Couleurs : Quelle palette de couleurs donner à la marque ? (Par exemple : Le bleu pour la confiance, le rouge pour l'urgence).
  • La Typographie : Quelle police d'écriture utiliser ? (Est-ce lisible ? Est-ce moderne ?).
  • Les Boutons : Quelle forme ? Où les placer ? Quelle couleur quand on passe la souris dessus ?
  • Les Icônes : Le style des petits symboles (maison, loupe, panier).
  • L'Espacement : L'espace entre les textes et les images pour que ce ne soit pas trop encombré.
  • Les Images : Le choix des photos et des illustrations.
Pourquoi c'est important ?

Une bonne UI donne une première impression positive. Si un site est moche ou mal organisé visuellement, l'utilisateur va partir avant même de tester si les fonctionnalités sont bonnes. L'UI inspire la confiance.

1. Le Dark Mode Évolué

L'obscurité élégante

Ce n'est plus juste une option, c'est la norme. Mais le dark mode moderne a évolué.


  • OLED Blacks : Utilisation de noir pur (#000000) sur les écrans OLED pour économiser la batterie.
  • Contraste Élevé : On abandonne le gris foncé pour des blancs éclatants sur fond noir très sombre.
  • Atmosphère : Crée une sensation de premium et de luxe (comme Netflix ou Spotify).
2. Glassmorphism

L'effet Verre

Inspiré par iOS et Windows 11, ce style utilise la transparence et le flou.


CSS backdrop-filter Blur Couches

  • Flou d'arrière-plan : Le contenu derrière l'élément est flouté, créant une sensation de profondeur.
  • Border Subtile : Des bordures blanches fines et translucides pour simuler le reflet sur le verre.
  • Usage : Parfait pour les menus flottants ou les cartes.
3. Grilles Bento

La modularité

Populaireisé par Apple (iOS widgets), ce style organise le contenu en boîtes rectangulaires.


  • Structure claire : Permet de présenter beaucoup d'informations de manière organisée.
  • Responsive : S'adapte parfaitement aux mobiles et aux grands écrans.
  • Esthétique : Donne un aspect soigné et technique aux dashboards.
4. Micro-interactions

Le site qui "vit"

L'interface n'est plus statique. Elle réagit à chaque geste de l'utilisateur.


  • Feedback Immédiat : Un bouton qui "écrase" au clic.
  • Guidage : Une animation qui montre où scroller.
  • Joystick : Sur mobile, les gestes (swipe) sont accompagnés d'animations fluides.
5. Typographie Géante & 3D

L'impact visuel

  • Typographie Massive : Des titres qui prennent 50% de l'écran. Le retour des polices "Serif" pour le côté éditorial.
  • Éléments 3D : Intégration d'objets 3D interactifs directement dans le web.
6. Conclusion

L'avenir

Les tendances évoluent vite, mais le but reste le même : l'expérience utilisateur (UX).


Que ce soit le verre, le noir ou le 3D, tout sert à rendre la navigation plus intuitive et plus agréable.
